# Hong Kong Corruption & Security Analysis

## Project Description
This project analyses the relationship between perceived corruption and security-related behaviour using World Values Survey data from Hong Kong (~2000 observations). 
- Dataset source:  
https://www.worldvaluessurvey.org/WVSDocumentationWV7.jsp

The analysis focuses on identifying factors influencing subjective feelings of security and evaluating the role of corruption perception in shaping behavioural decisions.

---

## Project Objective
- Assess whether perceived corruption influences security-related behaviour
- Identify the impact of demographic factors
- Analyse the relationship between confidence in the police and perceived security

---

## Data
- World Values Survey (Wave 7)
- Hong Kong
- ~2000 observations

---

## Analytical Methods
- Binary Logistic Regression  
- Multiple Linear Regression  
- Moderation Analysis  
- Statistical Inference  

---

## Key Findings
- Higher perceived corruption increases the likelihood of precautionary security behaviour (+10% odds)
- Demographic factors influence corruption perception
- Female respondents were more likely to demonstrate security-related precautionary behaviour
- Confidence in the police was not found to have a statistically significant moderating effect

---

## Business & Policy Relevance
The findings demonstrate how subjective perceptions of corruption can influence feelings of security and behavioural decision-making.

This type of analysis can support:
- public sentiment analysis
- risk perception analysis
- data-driven policy evaluation
- institutional trust research
- behavioural analytics
